# Website
Using HTML and CSS, I created a responsive website to display information about the company, allow the user to fill out a form to contact the company, show the history of the company and why it was started, promote package deals and pricing, tell the user what services the company offers and their individual pricing, show the active hours of the company with a table, give testemonials and allow the user to watch an informational video about the comany, meet the staff of the company and see some information about them, and provide links to related fields.
